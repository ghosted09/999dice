# 999dice On Terminal
999dice Bot For Termux Or Linux With Python2

## How to Run On Termux 
```bash
pkg install python2
pkg install curl
pkg install termux-api
pip2 install colorama
pip2 install requests
pip2 install bs4
pkg update
pkg upgrade
```

## How To Run On Linux
```bash
apt-get install python2
apt-get install curl
pip2 install colorama
pip2 install requests
pip2 install bs4
apt-get update
apt-get upgrade
```

## Usage
```bash
run Bot ( python2 999dice.py )

Stop Bot ( ctrl + 4 )
```
